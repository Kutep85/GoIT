Для задачки из Люксофта был создан класс FileOperator, в котором создается и заполняется текстовый файл, и считается вся статистика.
Класс FileRunner запускает методы из вышеописанного класса. 
Для работы с программой, необходимо только запустить FileRunner, и ввести количество слов, которое будет сгенерировано в текстовом файле.
Статистика будет выведена в консоль построчно.
